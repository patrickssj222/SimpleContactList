# Simple Contact List
##### Author: Yun Hao

### Goal
A simple contact app with authentication to view, add, edit, and delete contacts

### Prerequisite
XCode version > v12.2
### Usage Guide
```
Install dependencies:
    npm install

For iOS only:
    npm run pod-install

Run on simulation device:
    npm run start
    
    ios:    
    npm run ios
    
    android:
    npm run android  
```
